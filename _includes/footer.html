This project lovingly uses Github: Projects for project management, Actions for automation, and Pages for hosting.

These javascript/css pages use JQuery, Bootstrap, D3, underscore, D3-scale-radial, d3-array, and d3-geo-projection libraries.

These datasets are created from a variety of sources with R and Sparql in RStudio using tidyverse, SuperExactTest, jsonlite, magrittr, glue, qdapRegex, httr, lubridate, rvest, janitor, naniar, UpSetR, ggthemes, scales, numform.
